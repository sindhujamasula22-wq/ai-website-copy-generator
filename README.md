# AI Website Copy Generator for Local Businesses

## Task 1 — Future Interns Internship Programme

## Overview
An AI-generated website copy set for a local café business, created using structured prompts with Claude AI. Includes Homepage, Services, and CTA sections.

## Business Type
Café / Restaurant — The Golden Brew Café

## Tools Used
- Claude AI / ChatGPT
- GitHub

---

## Prompts & Outputs

### Homepage Copy
**Prompt:**
Write homepage copy for a cozy café called The Golden Brew in a busy urban neighbourhood. Include: a headline with a clear value proposition, a 3-sentence brand intro, and 3 bullet points highlighting key benefits. Tone: warm, welcoming, and community-focused.

**Output:**
- Headline: "Where Every Cup Feels Like Coming Home."
- Brand intro + 3 benefit bullet points (see cafe_website_copy.html)

---

### Services / Menu Page
**Prompt:**
Write service/menu descriptions for a café called The Golden Brew. Include 4 items, each with a name, a 2-sentence description, and a price placeholder. Tone: friendly, sensory, and appetising.

**Output:**
- Signature Espresso, Butter Croissant, Avocado Toast, Golden Turmeric Latte

---

### CTA Sections
**Prompt:**
Write 3 CTA sections for The Golden Brew café website: one for table booking, one for contact/enquiry, and one for a loyalty programme sign-up. Each should have a headline, 1-line subtext, and button text. Tone: warm and action-oriented.

**Output:**
- Book a Table / Get in Touch / Join Free (see cafe_website_copy.html)
